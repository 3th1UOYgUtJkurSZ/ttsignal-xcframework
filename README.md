# ttsignal-release

Binary distribution channel for the [ttsignal](https://github.com/3th1UOYgUtJkurSZ/ttsignal) QUIC signaling SDK.

Releases here host the prebuilt iOS xcframework (`ttsignal-swift.zip`) consumed by Swift Package Manager via `.binaryTarget(url:checksum:)`. See the parent repo for source.
